    # 介绍

    Moo-CSS (Moo, [mu:]哞~)是结合BEM、SMACSS、MCSS、OOCSS等写法思想，利用css modules及CSS in JS工具，根据实际需求整理总结而成的一套CSS写法规则策略，适用于大中小型web项目样式开发，旨在提高开发和维护效率。

    # 核心

    组件化+面向对象。
