# Moo-CSS Mobile demo(Zepto|VueJS)

## install & run

``` sh 
npm i && npm run start
```

### visit

chrome mobile pattern:

- `http://localhost:3000/mobileIndex.html`，原生
- `http://localhost:3000/mobileNavs.html`，vue
